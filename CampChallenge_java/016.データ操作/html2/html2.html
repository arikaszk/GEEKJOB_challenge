HTMLのタグ

他の多くのウェブページと同じく、このページもHTMLで作られています。
このページがどのように作られているかというと……

こんな感じ！

今見えているページとはだいぶ違うのがお分かりいただけると思います。
文章の前後に付いているこの青い文字。これがHTMLではかなり重要な『タグ』です。

タグとは、簡単に言うと『文字に役割を与えてあげる印』です。
たとえばページのタイトルであったり、見出しであったり、段落であったり。
同じ文章の中にも様々な役割がありますが、ただただ文字として打ってしまうと、 コンピューターは全て同じものとして認識してしまいます。

そこでそれぞれに「あなたはタイトルね」「あなたは見出しね」など、 役割を与えてあげるのがタグなのです！

タグの使い方は至ってシンプルです。

<開始タグ>印をつけたい文字<終了タグ>

このように印をつけたい文字の前と後ろに、開始タグと終了タグを付けてあげれば良いのです。
また、開始タグと終了タグは非常に似通っており、終了タグには『/（スラッシュ）』を付けるだけです。

【例1】見出し

<h1>HTMLについて</h1>

これで「HTMLについて」という文字に<h1>という役割が与えられました
<h1>のhというのは『heading（見出し）』という意味です

見出しにも種類があり、

これはh1の見出しです

これはh2の見出しです

これはh3の見出しです

これはh4の見出しです

これはh5の見出しです
これはh6の見出しです

このようにhの後に続く数字が大きくなるに連れて、文字のサイズが小さくなっていきます。

【例2】段落

<p>今回は『HTML』について学習したいと思います！</p>

これで「今回は『HTML』について学習したいと思います！」に<p>という役割が与えられました
<p>のpというのは『paragraph』のpです
文章の長さに規定はなく、開始タグから終了タグまでの間の文章が丸ごと一つの段落となります

上に紹介したふたつが、HTMLでもよく使用されるタグです。
他にもいくつかご紹介します。

タグ名 タグの役割
html 文書の中のどこからどこまでの部分がHTMLかを表す
head そのウェブページの情報を書き込む場所を表す
ここに書き込まれた内容はウェブページ上には表示されない
body ウェブページの画面に表示される場所を表す
li 箇条書きにする
※<ul>タグで囲む必要がある
br 改行をする
※終了タグは必要ない

実際に書いてみる

ここまでHTMLの基本を説明してきましたが、今度は実際に簡単なコードを書いてみたいと思います。

用意するもの

  • パソコン
  • テキストエディタ（Macならテキストエディット、Windowsならメモ帳でOK！）
  • インターネットブラウザ（Safari、Google Crome、Fire Fox、Internet Explorerなど）

まずはテキストエディタに以下の文章をコピペします

<!DOCTYPE html>
<html>
<head>
  <title>HTMLを学ぶ</title>
</head>
<body>
  <h1>HTMLの使い方</h1>
</body>
</html>

あとは、<body>と</body>の間に好きな文字を打ち込んでみましょう！

<!DOCTYPE html>
<html>
<head>
  <title>HTMLを学ぶ</title>
</head>
<body>
  <h1>HTMLの使い方</h1>
  <p>これで簡単なウェブページの作成ができます！</p>
  <ul>
    <li>ウェブページは『HTML』と『CSS』でできている</li>
    <li>HTMLではタグで文字を囲うことで印をつける</li>
    <li>終了タグが必要ないタグもある</li>
  </ul>
</body>
</html>

これを[ファイル]→[保存]から拡張子を『.html』にして保存
あとはそのファイルをブラウザを使用して開くだけです！
